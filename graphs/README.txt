Graphs module

Purpose:
- Convert directory trees into node-link JSON for frontend use.
- Keep output capped for fast rendering in hackathon demos.

How to use:
1) Start backend dependencies in the same Python environment.
2) Run from prototype-12h/backend:
   python ..\graphs\export_graph_snapshot.py
3) Enter a local repository path.
4) Result is written to graphs/sample_graph.json.
